----  
  
起動ステータス  
  
LED点滅2回  
→ SRAMヒープメモリから起動  
  
LED点滅2回 + LED点滅3回  
→ PSRAMヒープメモリから起動  
  
LED点滅2回 + LED点滅3回 + LED点滅1～5回  
→ PSRAMヒープメモリのテストに失敗したのでSRAMヒープメモリから起動
